# WizardWnd

MFC로 제작된 설치 마법사 스타일의 자식 다이얼로그 전환 테스트 예제입니다.

## 기능 설명

* '이전 페이지로' 버튼이나 '다음 페이지로' 버튼을 클릭해서 현재 표시되는 자식 비정형 다이얼로그를 변경할 수 있습니다.
* 첫 번째 페이지에서 피연산자 두 개를 입력한 다음 두 번째 페이지에서 연산자를 선택하고 나서 세 번째 페이지로 이동하면 연산 결과를 보여줍니다.
